# vehicles-web-app
"Web application for vehicle sales data analysis"

## Descripción del Proyecto
Esta aplicación web interactiva permite a los usuarios explorar y analizar datos de anuncios de venta de vehículos usados a través de visualizaciones dinámicas.

## Para qué sirve
La aplicación sirve para:
- Analizar tendencias en el mercado de vehículos usados
- Visualizar la distribución del kilometraje de los vehículos
- Explorar la relación entre el kilometraje y el precio de venta

## Funcionalidad
La aplicación proporciona:
- **Histograma interactivo**: Muestra la distribución del odómetro de los vehículos
- **Gráfico de dispersión**: Visualiza la relación entre kilometraje (odómetro) y precio
- **Interfaz intuitiva**: Botones interactivos para generar visualizaciones bajo demanda

## Tecnologías Utilizadas
- Python
- Streamlit
- Plotly Express
- Pandas
